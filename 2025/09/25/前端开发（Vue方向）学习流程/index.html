前端开发（Vue方向）学习流程

发表于2025-09-25|更新于2025-10-02

🛠 前端开发（Vue方向）学习流程

1. Web 基础（三大件）

这是所有前端的根基。

  • HTML
    • 常用标签、语义化标签、表单与输入控件、meta 标签与 SEO 基础
  • CSS
    • 选择器与优先级、盒子模型、定位、布局（Flex、Grid）、动画、响应式设计、预处理器
  • JavaScript
    • 变量与作用域、数据类型、函数与闭包、原型与继承、ES6+ 特性、DOM 操作、异步编程（Promise/async/await）、模块化、JSON 与本地存储

2. 前端进阶

  • 浏览器原理
    • 事件循环、宏任务与微任务
    • 渲染机制（回流、重绘、性能优化）
    • 跨域与解决方案
  • ES6+
    • 类与模块化、Set/Map、解构赋值、模板字符串、async/await
  • TypeScript
    • 类型系统（基本类型、接口、泛型）
    • Vue3 中的 TS 应用

3. Vue 核心

  • Vue 基础
    • 响应式原理、模板语法、组件化、计算属性与侦听器、生命周期、组合式 API
  • Vue 进阶
    • 组件通信、插槽、自定义指令、依赖注入
    • Vue Router（路由与导航守卫）
    • Pinia（状态管理）
    • Vue3 新特性（Teleport、Suspense）

4. 前端工程化

  • 包管理工具：npm、pnpm、yarn
  • 构建工具：Vite / Webpack
  • 代码规范：ESLint、Prettier、Git 提交规范
  • CSS 工程化：CSS Module、TailwindCSS、Element Plus
  • API 调用：Axios、RESTful API、GraphQL
  • 单元测试：Vitest / Jest
  • 性能优化：路由懒加载、Tree-Shaking、图片懒加载、代码分割

5. 软实力

  • Git & GitHub/Gitee：基础命令、分支管理
  • Linux 基础：常见命令、SSH 部署
  • 简历与面试准备：Vue 响应式原理、事件循环、跨域、性能优化等常见题
